# Projeto 1 do curso de Python

O projeto consiste em um jogo em que será apresentada uma expressão matemática podendo ser de soma, subtração ou multiplicação, dessa forma o usuário acumula pontos ao acertar o valor correto da expressão

Nesse projeto há a utilização de conceitos de programação orientada a objetos e o uso da biblioteca randint que gera números aleatórios dentro de um limite especificado
